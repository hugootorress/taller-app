Ayuda y Acerca de

TallerMate

Versión: 1.0

Desarrollada por: Hugo Torres

TallerMate es una aplicación web para la gestión integral de talleres mecánicos.
Permite administrar clientes, vehículos, reparaciones, piezas, facturación y comunicación por correo electrónico.
Incluye control de IVA configurable, generación de facturas en PDF y panel de administración para mecánicos.

Soporte: Para dudas o incidencias, contacta con el desarrollador.
Fecha: Junio 2025

  • Gestión de mecánicos, clientes y vehículos
  • Control de reparaciones y piezas
  • Facturación automática con IVA
  • Envío de correos al taller
  • Panel de administración para usuarios con rol de administrador
